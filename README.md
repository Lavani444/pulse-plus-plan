## What it does

PulsePlusPlan is a full-stack scheduling application that learns your behaviour patterns and automatically distributes your time across the areas that matter — work, fitness, nutrition, and personal development. Instead of a static calendar, it adapts to how you actually use your time and surfaces smarter suggestions over time.

## Features

- **Intelligent time blocking** — analyses user behaviour to auto-distribute daily time slots
- **Priority balancing algorithm** — adapts scheduling suggestions based on historical patterns and goal metrics
- **Workout tracker** — exercise library, progress visualisation, and session logging
- **Meal & macro planner** — integrated nutrition tracking with daily macro targets
- **Unified dashboard** — all life categories in one view with performance analytics
- **Responsive design** — works across desktop and mobile via Tailwind CSS

## Getting started

```bash
# Clone the repo
git clone https://github.com/Lavani444/pulse-plus-plan.git
cd pulse-plus-plan

# Install dependencies
npm install

# Start the development server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

> **Note:** You'll need to add a `.env` file with your Firebase config keys. See `.env.production` for the required variable names.
